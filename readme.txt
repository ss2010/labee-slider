Easy to use WordPress slider plugin. Create SEO optimised Responsive Slides.

== Description ==
Labee Slider is a responsive slider plugin. It has settings with Background image, Button Text, Button URL, Boxed Styling, Positons-left or right. You can set Videos (Youtube, Vimeo) also. Slider Controllers automatically increased with the number of Slider Posts.

== Installation ==
1.Go to the Plugins Menu in WordPress
2.Search for \"Labee Slider\"
3.Click \"Install\"

== Frequently Asked Questions ==
How to Use?
To set up a Slider Go to WP Admin Dashboard. See a Menu named \"Slider\" is a post type, click Add New. Slider supports- Title, Editor, Thumbnail.
